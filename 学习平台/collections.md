
# **Tips**
1. 使用Google或者bing搜索，然后输入关键词，如【Java学习】【后端学习】等关键字，然后点开链接开学。
2. Github上搜索关键字
3. 有条件的可以加入一些感兴趣并且能给你提升的社区，比较经典的就是【知识星球】，一般免费的部分都有对付费部分的详细的介绍和描述，可以根据实际情况选择
4. 东西都是相通的，还是得自己整理形成属于自己的体系，要秉持的有输入适当输出的原则，看会懂了是一码事，但是能不能用自己的话讲出来又是一码事，所以一定要动手写代码，动手实践。再就是积累的过程强调的就是一个重复，重复，重复的过程。
5.
