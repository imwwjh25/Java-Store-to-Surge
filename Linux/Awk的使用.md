1. **提取列**：比如取文件的第 1 列和第 3 列（默认空格或制表符分隔）


   ```bash
   awk '{print $1, $3}' file.txt
   ```

   

2. **按条件筛选行**：比如筛选第 2 列大于 100 的行


   ```bash
   awk '$2 > 100 {print}' data.txt
   ```

   

3. **自定义分隔符**：比如用逗号分隔，取第 2 列

   ```bash
   awk -F ',' '{print $2}' csvfile.txt
   ```

   

4. **统计计算**：比如计算第 3 列的总和


   ```bash
   awk '{sum += $3} END {print sum}' numbers.txt
   ```

   

`awk` 还支持变量、循环、条件判断等编程特性，适合处理复杂的文本分析任务。如果有具体场景需要处理，可以告诉我，我可以给出更具体的用法示例～
