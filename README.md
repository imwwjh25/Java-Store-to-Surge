## Ps

1. 现在AI确实能很大程度的提效，比如遇到一个问题都省去了搜索引擎搜索的过程，直接交给AI就行，但是AI的回答有时候会有点问题，需要自己甄别。但第一个阶段是【量】的一个过程，量变才引起质变，所以很多问题都是AI生成的。
2. 对于想深入系统的学习和掌握的部分，还是要老三样【书籍，课程，动手】
